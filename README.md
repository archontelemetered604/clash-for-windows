## Overview
Clash for Windows is a powerful and user-friendly graphical client for Clash core. It supports rule-based proxying with advanced features including Shadowsocks, V2Ray, Trojan, Hysteria, VMess, SSR and many other protocols. The application provides an intuitive dashboard, system proxy mode, TUN mode, global and rule-based routing, and excellent stability. Perfect for users who need a reliable and feature-rich proxy client on Windows 11.

## Installation
### For Regular Users
1. Download and extract the archive
2. Run `Clash for Windows.exe` as Administrator
3. Follow the setup guide
4. Import your subscription or configuration file
5. Select proxy mode (Rule / Global / Direct)
6. Enable System Proxy or TUN mode and start browsing

## Troubleshooting
| Problem | Solution |
|--------------------------------|-------------------------------------------------------|
| App doesn't start | Run as Administrator and disable antivirus temporarily |
| Configuration not loading | Check subscription link and update config |
| No internet after enabling proxy | Switch to Rule mode or check TUN settings |
| High CPU / memory usage | Update to latest version 2026 |
| Connection unstable | Try different nodes and enable fake-ip |
| Windows 11 compatibility error | Update Windows and run the setup guide again |
| Dashboard not opening | Restart the application |
| Download / update failed | Use the latest release and check your internet connection |
